# Is Array Sorted

## Problem Statement
Write a program that takes an integer array `a` of length `n` and prints `1` if the array is sorted in ascending order; otherwise, it prints `0`.

## Example 1

### Input:
```
5
1 2 3 4 5
```

### Output:
```
1
```

## Example 2

### Input:
```
5
5 3 2 4 1
```

### Output:
```
0
```
