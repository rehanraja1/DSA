# Left Rotate an Array by 1 Place

## Problem Statement
Given an array of integers, your task is to rotate the array to the left by one position.

## Example

### Input:
```
arr = [1, 2, 3, 4, 5]
```

### Output:
```
[2, 3, 4, 5, 1]
```

### Input:
```
arr = [10, 20, 30, 40]
```

### Output:
```
[20, 30, 40, 10]
```
