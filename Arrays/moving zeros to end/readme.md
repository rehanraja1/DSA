# Move Zeros to End

## Problem Statement
You are given an array of integers. Your task is to write a program that moves all the zeros in the array to the end while maintaining the order of the non-zero elements.

## Example

### Input:
```
[0, 1, 0, 3, 12]
```

### Output:
```
[1, 3, 12, 0, 0]
```

### Input:
```
[0, 0, 1, 2, 0, 4]
```

### Output:
```
[1, 2, 4, 0, 0, 0]
```
