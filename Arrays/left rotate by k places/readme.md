# Left Rotate an Array by K Places

## Problem Statement
Given an array of integers, your task is to rotate the array to the left by `k` positions.

## Example

### Input:
```
arr = [1, 2, 3, 4, 5]
k = 2
```

### Output:
```
[3, 4, 5, 1, 2]
```

### Explanation:
Rotating the array `[1, 2, 3, 4, 5]` to the left by `2` positions results in `[3, 4, 5, 1, 2]`.
