# Binary Tree Paths

## Problem Statement
Given the root of a binary tree, your task is to find all paths from the root to the leaf nodes. Return a list of lists where each sublist shows the nodes in a path from the root to a leaf in the tree.

## Example 1:

### Input:
```
       1
      / \
     2   3
    / \   \
   4   5   6
```

### Output:
```
[
  [1, 2, 4],
  [1, 2, 5],
  [1, 3, 6]
]
```
